# Distributed Spotify
> Un diseño distribuido de una aplicación como Spotify

### Correr el proyecto

El archivo `server.py` ejecuta una instancia de servidor y el `docker-compose.yml` levanta 4 instancias.
```bash
docker compose up -d
```
